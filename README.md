# Code Similarity Hub

A beautiful redirect website for two Code Similarity tools.

## Links
- CodeGNN Clone Detection → https://huggingface.co/spaces/vedantk23/CodeSimilarity
- Assignment Code Checker → https://codesimilarity.streamlit.app

## Deploy on Vercel

1. Push this folder to GitHub
2. Go to vercel.com → New Project
3. Import your repository
4. Deploy (Vercel will auto-detect it as static site)
